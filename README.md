# IndexTTS-2 音色资源合集

本仓库用于整理与备份一批中文语音克隆/配音参考音色文件，便于检索与分类管理。

## 仓库概览

- 资源类型：`wav`、`mp3` 为主，少量图片与文本文件
- 文件数量：约 `1148` 个（排除 `.git`）
- 主要音频格式：`wav`（598）、`mp3`（531）

## 目录结构

```text
.
├── 不同年龄人群音色/
├── 不同情绪音色/
├── 克隆参考音色/
├── 音色整理合集/
├── 马宝国/
└── 更多资源.txt
```

## 目录体积（近似）

- `不同年龄人群音色/`：约 221M
- `不同情绪音色/`：约 46M
- `克隆参考音色/`：约 256M
- `音色整理合集/`：约 89M
- `马宝国/`：约 5.6M

## 使用建议

1. 按目录语义筛选目标音色，再做二次试听与标注。
2. 训练或克隆前建议统一采样率、声道与响度，降低数据分布差异。
3. 对高频使用音色可单独维护索引清单（如 `csv`/`json`）提升检索效率。
